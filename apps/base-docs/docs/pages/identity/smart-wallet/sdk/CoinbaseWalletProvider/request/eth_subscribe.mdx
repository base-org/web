# eth_subscribe

Defined in the [Ethereum JSON-RPC Specification](https://ethereum.org/en/developers/docs/apis/json-rpc/)

> Creates a subscription to specific events. The node will return a subscription id.

## Parameters

```ts
type SubscribeParams = [string, object?]
```

1. Subscription name
2. Optional subscription parameters

Supported subscription types:

- `newHeads`: New block headers
- `logs`: Logs matching a filter object
- `newPendingTransactions`: New pending transactions
- `syncing`: Syncing status updates

## Returns

`string`

Subscription ID. This ID is used with `eth_unsubscribe` to cancel the subscription.

## Errors

| Code | Message                   |
| ---- | ------------------------- |
| 4200 | Unsupported method        |
| 4001 | User rejected the request |
