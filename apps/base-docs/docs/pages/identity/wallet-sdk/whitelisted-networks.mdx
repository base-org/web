Coinbase Wallet supports **all** EVM-compatible L1/L2 networks. Certain networks are whitelisted, allowing Coinbase Wallet to automatically detect all assets held in the user's wallet.

## Main networks

- Arbitrum
- Avalanche C-Chain
- Base
- Binance Smart Chain
- Ethereum
- Fantom Opera
- Gnosis
- Optimism
- Polygon
