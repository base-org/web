# eth_decrypt

Defined in the [MetaMask JSON-RPC API](https://docs.metamask.io/wallet/reference/json-rpc-api/)

> Requests that MetaMask decrypt the specified encrypted message. The message must have been encrypted using the public encryption key of the specified address.

## Parameters

```ts
interface DecryptParams {
  /**
   * The encrypted data to decrypt.
   */
  data: string
  /**
   * The address whose encryption key should be used to decrypt the data.
   */
  address: string
}
```

## Returns

`string`

The decrypted message.

## Errors

| Code | Message                                       |
| ---- | --------------------------------------------- |
| 4001 | User rejected the request                     |
| 4200 | Unsupported method                            |
| 4100 | The requested account has not been authorized |
| 4300 | Decryption failed                             |
