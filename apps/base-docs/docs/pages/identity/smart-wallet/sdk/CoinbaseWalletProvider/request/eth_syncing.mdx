# eth_syncing

> Returns an object with data about the sync status or false.

## Returns

`object` or `boolean`

Returns an object with sync status data or `false` when not syncing:

```typescript
interface SyncStatus {
  startingBlock: string; // The block at which the import started
  currentBlock: string; // The current block, same as eth_blockNumber
  highestBlock: string; // The estimated highest block
}
```

### Example when syncing

```json
{
  "startingBlock": "0x384",
  "currentBlock": "0x386",
  "highestBlock": "0x454"
}
```

### Example when not syncing

```
false
```

## Errors

This method does not typically return errors.
