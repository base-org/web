# eth_getFilterChanges

Defined in the [Ethereum JSON-RPC Specification](https://ethereum.org/en/developers/docs/apis/json-rpc/)

> Polling method for a filter, which returns an array of logs which occurred since the last poll.

## Parameters

```ts
type GetFilterChangesParams = [string]
```

An array containing a single string: the filter ID.

## Returns

`Array<FilterLog | string>`

Array of log objects, or an array of transaction hashes depending on the filter type:

- For filters created with `eth_newBlockFilter`: Block hashes
- For filters created with `eth_newPendingTransactionFilter`: Transaction hashes
- For filters created with `eth_newFilter`: Log objects

## Errors

| Code | Message            |
| ---- | ------------------ |
| 4200 | Unsupported method |
| 4300 | Filter not found   |
