# wallet_requestPermissions

Defined in [EIP-2255](https://eips.ethereum.org/EIPS/eip-2255)

> Requests permissions from the user. This method displays a user interface where the user can approve or reject the requested permissions.

## Parameters

```typescript
interface RequestedPermissions {
  [methodName: string]: {}; // Currently, each permission object is empty
}
```

### Example

```json
{
  "eth_accounts": {}
}
```

## Returns

`Array<PermissionObject>`

Returns an array of permission objects that have been granted by the user.

```typescript
interface PermissionObject {
  parentCapability: string; // The name of the method that this permission allows
  invoker: string; // The origin of the requesting site
  caveats?: PermissionCaveat[]; // Optional caveats restricting the permission
}
```

### Example

```json
[
  {
    "parentCapability": "eth_accounts",
    "invoker": "https://example.com",
    "caveats": [
      {
        "type": "restrictReturnedAccounts",
        "value": ["0x0c54fccd2e384b4bb6f2e405bf5cbc15a017aafb"]
      }
    ]
  }
]
```

## Errors

| Code | Message                   |
| ---- | ------------------------- |
| 4001 | User rejected the request |
