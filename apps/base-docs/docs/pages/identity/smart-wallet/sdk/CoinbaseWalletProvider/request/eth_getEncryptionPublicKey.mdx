# eth_getEncryptionPublicKey

Defined in the [MetaMask JSON-RPC API](https://docs.metamask.io/wallet/reference/json-rpc-api/)

> Requests the public encryption key from the user. Returns a public encryption key that can be used with `eth_decrypt`.

## Parameters

```ts
type GetEncryptionPublicKeyParams = [string]
```

An array with a single item: the Ethereum address whose encryption key should be retrieved.

## Returns

`string`

The public encryption key, which can be used to encrypt messages intended for the requested account.

## Errors

| Code | Message                                       |
| ---- | --------------------------------------------- |
| 4001 | User rejected the request                     |
| 4200 | Unsupported method                            |
| 4100 | The requested account has not been authorized |
