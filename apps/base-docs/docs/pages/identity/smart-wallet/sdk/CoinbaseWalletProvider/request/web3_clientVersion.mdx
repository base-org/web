# web3_clientVersion

Defined in the [Ethereum JSON-RPC Specification](https://ethereum.org/en/developers/docs/apis/json-rpc/)

> Returns the current client version.

## Returns

`string`

The current client version as a string.

## Example

```
"MetaMask/v10.8.1"
```

## Errors

| Code | Message            |
| ---- | ------------------ |
| 4200 | Unsupported method |
