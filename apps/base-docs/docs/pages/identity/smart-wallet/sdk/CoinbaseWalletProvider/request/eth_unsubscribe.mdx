# eth_unsubscribe

Defined in the [Ethereum JSON-RPC Specification](https://ethereum.org/en/developers/docs/apis/json-rpc/)

> Cancels an existing subscription so that no further events are sent.

## Parameters

```ts
type UnsubscribeParams = [string]
```

An array containing a single string: the subscription ID previously returned from `eth_subscribe`.

## Returns

`boolean`

`true` if the subscription was successfully cancelled, `false` if it wasn't found.

## Errors

| Code | Message            |
| ---- | ------------------ |
| 4200 | Unsupported method |
