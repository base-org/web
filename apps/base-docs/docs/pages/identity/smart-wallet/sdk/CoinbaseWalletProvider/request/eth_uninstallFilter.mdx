# eth_uninstallFilter

Defined in the [Ethereum JSON-RPC Specification](https://ethereum.org/en/developers/docs/apis/json-rpc/)

> Uninstalls a filter with the given ID. Should always be called when a filter is no longer needed.
> Additionally, filters timeout when they aren't requested with `eth_getFilterChanges` for a period of time.

## Parameters

```ts
type UninstallFilterParams = [string]
```

An array containing a single string: the filter ID.

## Returns

`boolean`

`true` if the filter was successfully uninstalled, `false` if it wasn't found.

## Errors

| Code | Message            |
| ---- | ------------------ |
| 4200 | Unsupported method |
