# wallet_registerOnboarding

Defined in the [MetaMask JSON-RPC API](https://docs.metamask.io/wallet/reference/json-rpc-api/)

> This method is used by the MetaMask onboarding library to track which sites have initiated onboarding. Used to provide insight to MetaMask Flask developers about the onboarding process.

`boolean`

Returns `true` if the call was successful.
