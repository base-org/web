# wallet_swapAsset

Defined in the [MetaMask JSON-RPC API](https://docs.metamask.io/wallet/reference/json-rpc-api/)

> Opens a view where users can swap assets on MetaMask Swaps, with the provided parameters to pre-populate the UI.

## Parameters

```ts
interface SwapAssetParams {
  /**
   * Address of the asset to swap from.
   */
  fromAsset: string
  /**
   * Address of the asset to swap to.
   */
  toAsset: string
  /**
   * Amount of the fromAsset to swap, as a string denominated in the asset's smallest unit.
   */
  amount: string
  /**
   * Type of swap to perform. Currently only 'swapExactIn' is supported.
   */
  swapType: "swapExactIn"
}
```

## Returns

```ts
interface SwapAssetResult {
  /**
   * The transaction hash of the swap transaction.
   */
  transactionHash: string
}
```

## Errors

| Code | Message                   |
| ---- | ------------------------- |
| 4001 | User rejected the request |
| 4200 | Unsupported method        |
| 4300 | Swap failed               |
