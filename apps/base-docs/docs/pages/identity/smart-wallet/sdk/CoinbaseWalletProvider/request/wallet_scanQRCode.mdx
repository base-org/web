# wallet_scanQRCode

Defined in the [MetaMask JSON-RPC API](https://docs.metamask.io/wallet/reference/json-rpc-api/)

> Activates the device's camera to scan a QR code. Returns the scanned string if successful.

## Parameters

```ts
type ScanQRCodeParams = string | undefined
```

The parameter is an optional regex string to validate the QR code content.

## Returns

`string`

The content of the scanned QR code.

## Errors

| Code | Message                   |
| ---- | ------------------------- |
| 4001 | User rejected the request |
| 4200 | Unsupported method        |
| 4300 | QR code scan canceled     |
| 4301 | QR code scan failed       |
