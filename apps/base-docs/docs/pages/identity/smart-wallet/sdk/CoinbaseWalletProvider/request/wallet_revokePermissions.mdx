# wallet_revokePermissions

Defined in the [MetaMask JSON-RPC API](https://docs.metamask.io/wallet/reference/json-rpc-api/)

> Allows a site to revoke a permission previously granted by the user via `wallet_requestPermissions`.

## Parameters

```ts
interface RevokePermissionsParams {
  /**
   * The permissions to revoke.
   */
  permissions: PermissionConstraint[]
}

interface PermissionConstraint {
  /**
   * The name of the permission to revoke.
   */
  parentCapability: string
}
```

## Returns

`null`

## Errors

| Code | Message                                                   |
| ---- | --------------------------------------------------------- |
| 4001 | User rejected the request                                 |
| 4200 | Unsupported method                                        |
| 4100 | The requested account has not been authorized by the user |
